Windows Phone ( WP ) to wycofana rodzina mobilnych systemów operacyjnych opracowanych przez Microsoft dla smartfonów jako zastępczy następca Windows Mobile i Zune. Windows Phone zawierał nowy interfejs użytkownika wywodzący się z języka projektowego Metro. W przeciwieństwie do Windows Mobile był skierowany głównie na rynek konsumencki, a nie na rynek korporacyjny.

Po raz pierwszy został uruchomiony w październiku 2010 roku wraz z systemem Windows Phone 7. Windows Phone 8 zastąpił go w 2012 roku, zastępując jądro Windows Phone 7 oparte na Windows CE jądrem Windows NT używanym przez wersje systemu Windows na komputery PC (a w szczególności dużą liczbę wewnętrznych komponentów z Windows 8 ). Z powodu tych zmian system operacyjny był niekompatybilny ze wszystkimi istniejącymi urządzeniami z Windows Phone 7, chociaż nadal obsługiwał aplikacje pierwotnie opracowane dla Windows Phone 7. W 2014 roku Microsoft wydał aktualizację Windows Phone 8.1, która wprowadziła asystenta wirtualnego Cortana oraz obsługę środowiska wykonawczego systemu Windows do tworzenia aplikacji międzyplatformowych między komputerami z systemem Windows i telefonami z systemem Windows Phone. W 2015 roku firma Microsoft wypuściła system Windows 10 Mobile. Chociaż Microsoft porzucił wówczas markę Windows Phone, aby bardziej skupić się na synergii z systemem Windows 10 na komputery PC, z technicznego punktu widzenia nadal był kontynuacją linii Windows Phone, a dla wybranych urządzeń z systemem Windows Phone 8.1 wydano aktualizacje.
