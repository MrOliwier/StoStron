Windows 8

System ten to główne wydanie systemu operacyjnego Windows NT opracowanego przez firmę Microsoft . Został wydany do produkcji 1 sierpnia 2012 roku; został następnie udostępniony do pobrania za pośrednictwem MSDN i TechNet 15 sierpnia 2012 r., a później w sprzedaży detalicznej 26 października 2012 r. Windows 8 wprowadził poważne zmiany w platformie systemu operacyjnego i interfejsie użytkownika, mające na celu poprawę komfortu użytkowania na tabletach, gdzie Windows konkurował teraz z mobilnymi systemami operacyjnymi, w tym Androidem i iOS. W szczególności zmiany te obejmowały powłokę systemu Windows zoptymalizowaną pod kątem obsługi dotykowej, opartą na języku projektowym Metro firmy Microsoft i ekranie startowym, nową platformę do tworzenia aplikacji z naciskiem na wprowadzanie danych za pomocą ekranu dotykowego, integrację z usługami online oraz Sklep Windows, sklep internetowy dystrybucjado pobierania i kupowania nowego oprogramowania oraz nowy skrót klawiaturowy do zrzutów ekranu. Wiele z tych funkcji zostało przejętych z Windows Phone. Windows 8 dodał obsługę USB 3.0, dysków twardych Advanced Format, komunikacji bliskiego zasięgu i przetwarzania w chmurze. Wprowadzono dodatkowe funkcje bezpieczeństwa, takie jak wbudowane oprogramowanie antywirusowe, integracja z usługą filtrowania phishingu Microsoft SmartScreen oraz obsługa UEFI Secure Boot na obsługiwanych urządzeniach. Windows 8 to pierwsza wersja systemu Windows obsługująca architekturę ARM pod szyldemZnakowanie Windowsa RT. W systemie Windows 8 usunięto obsługę procesorów innych niż SSE2 i urządzeń bez NX. Windows 8 został wydany z mieszanym przyjęciem krytycznym. Chociaż reakcja na ulepszenia wydajności, ulepszenia bezpieczeństwa i ulepszoną obsługę urządzeń z ekranem dotykowym była pozytywna, nowy interfejs użytkownika systemu operacyjnego był szeroko krytykowany za to, że może być mylący i trudny do nauczenia, zwłaszcza gdy jest używany z klawiaturą i myszą zamiast z ekran dotykowy. Pomimo tych niedociągnięć do stycznia 2013 roku sprzedano 60 milionów licencji Windows 8, a liczba ta obejmowała zarówno aktualizacje, jak i sprzedaż producentom OEM nowych komputerów. Firma Microsoft wypuściła system Windows 8.1 w październiku 2013 r., Odnosząc się do niektórych aspektów systemu Windows 8, które były krytykowane przez recenzentów i wczesnych użytkowników, oraz zawierała dodatkowe ulepszenia różnych aspektów systemu operacyjnego. System Windows 8 został ostatecznie zastąpiony przez system Windows 10 w lipcu 2015 r. Serwerowy odpowiednik, Windows Server 2012, obecnie otrzymuje aktualizacje zabezpieczeń do października 2023 r., a nieoficjalne metody zostały zastosowane w celu zastosowania tych poprawek do oryginalnego systemu Windows 8.

Windows 8.1 to wersja systemu operacyjnego Windows NT opracowana przez firmę Microsoft.

Jest to wersja systemu operacyjnego Windows NT opracowana przez firmę Microsoft. Został wypuszczony do produkcji 27 sierpnia 2013 r. I szeroko wprowadzony do sprzedaży detalicznej 17 października 2013 r., Około rok po wydaniu detalicznym swojego poprzednika, a zastąpił go system Windows 10 29 lipca 2015 r. Udostępniono system Windows 8.1 do pobrania za pośrednictwem witryn MSDN i Technet oraz jako bezpłatne uaktualnienie dla detalicznych kopii użytkowników systemów Windows 8 i Windows RT za pośrednictwem Sklepu Windows. Wersja serwerowa , Windows Server 2012 R2, został wydany 18 października 2013 r. Windows 8.1 miał na celu odpowiedzieć na skargi użytkowników i recenzentów systemu Windows 8 w momencie premiery. Ulepszenia obejmują ulepszony ekran Start, dodatkowe widoki przyciągania, dodatkowe pakiety aplikacji, ściślejszą integrację OneDrive (dawniej SkyDrive), Internet Explorer 11 (IE11), ujednolicony system wyszukiwania oparty na Bing, przywrócenie widocznego przycisku Start na pasku zadań oraz możliwość przywrócenia poprzedniego zachowania otwierania pulpitu użytkownika przy logowaniu zamiast ekranu startowego. Windows 8.1 dodał także obsługę pojawiających się wówczas technologii, takich jak wyświetlacze o wysokiej rozdzielczości, drukowanie 3D, Wi-Fi Direct i przesyłanie strumieniowe Miracast, a także system plików ReFS. Windows 8.1 spotkał się z bardziej pozytywnym przyjęciem niż Windows 8, a krytycy chwalili rozszerzoną funkcjonalność dostępną dla aplikacji w porównaniu z Windows 8, integrację z OneDrive, ulepszenia interfejsu użytkownika oraz dodanie rozszerzonych samouczków dotyczących obsługi interfejsu Windows 8. Pomimo tych ulepszeń system Windows 8.1 był nadal krytykowany za nierozwiązywanie wszystkich problemów systemu Windows 8 (takich jak słaba integracja między aplikacjami w stylu Metro a interfejsem komputera stacjonarnego) oraz potencjalne konsekwencje dla prywatności wynikające z rozszerzonego korzystania z usług online. Oficjalne wsparcie głównego nurtu systemu Windows 8.1 zakończyło się 9 stycznia 2018 r., a rozszerzone wsparcie zakończy się 10 stycznia 2023 r., podczas gdy odpowiednik serwera, Windows Server 2012 R2, zakończył wsparcie głównego nurtu 9 października 2018 r. i rozszerzył wsparcie zakończy się 10 października 2023 r. Podstawowe wsparcie dla wersji Embedded systemu Windows 8.1 zakończyło się 10 lipca 2018 r., a rozszerzone wsparcie zakończy się 11 lipca 2023 r. Według stanu na październik 2022 r. 2,7% tradycyjnych komputerów PC z systemem Windows korzysta z systemu Windows 8.1, co plasuje go na 4. miejscu wśród najpopularniejszych systemów Windows od czasu , gdy wyprzedził go system Windows 11.
