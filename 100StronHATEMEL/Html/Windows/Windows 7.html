Windows 7 – wydanie systemu operacyjnego Windows firmy Microsoft, następca systemu Windows Vista.

Został wypuszczony do produkcji 22 lipca 2009 r. i stał się ogólnie dostępny 22 października 2009 r. Jest następcą systemu Windows Vista, wydanego prawie trzy lata wcześniej. Pozostał systemem operacyjnym do użytku na komputerach osobistych, w tym domowych i biznesowych komputerach stacjonarnych, laptopach, tabletach i centrach multimedialnych, a sam został zastąpiony w listopadzie 2012 r. przez Windows 8, nazwa obejmująca ponad trzy lata produktu. Do 9 kwietnia 2013 r. Oryginalna wersja systemu Windows 7 zawierała aktualizacje i wsparcie techniczne, po czym wymagana była instalacja dodatku Service Pack 1, aby użytkownicy mogli otrzymywać wsparcie i aktualizacje. Serwerowy odpowiednik Windows 7, Windows Server 2008 R2, został wydany w tym samym czasie. Ostatnia obsługiwana wersja systemu Windows oparta na tym systemie operacyjnym została wydana 1 lipca 2011 r., zatytułowana Windows Embedded POSReady 7. Rozszerzone wsparcie zakończyło się 14 stycznia 2020 r., ponad dziesięć lat po wydaniu systemu Windows 7, po czym system operacyjny przestał otrzymywać dalsze wsparcie. Obecnie dostępny jest program wsparcia dla przedsiębiorstw, zapewniający aktualizacje zabezpieczeń dla systemu Windows 7 przez okres do czterech lat od oficjalnego zakończenia jego eksploatacji. Jednak system Windows Embedded POSReady 7, ostatni wariant systemu Windows 7, nadal otrzymywał aktualizacje zabezpieczeń do października 2021 r. Windows 7 miał być stopniową aktualizacją systemu Microsoft Windows, mającą na celu rozwiązanie problemu słabego odbioru systemu Windows Vista przez krytyków przy jednoczesnym zachowaniu zgodności sprzętu i oprogramowania. W systemie Windows 7 kontynuowano ulepszenia interfejsu użytkownika Windows Aero, dodając przeprojektowany pasek zadań, który umożliwia przypinanie aplikacji, oraz nowe funkcje zarządzania oknami. Do systemu operacyjnego dodano inne nowe funkcje, w tym biblioteki, nowy system udostępniania plików HomeGroup oraz obsługę wprowadzania wielodotykowego. Dodano również nowe „ Centrum akcji ”, aby zapewnić przegląd informacji o bezpieczeństwie i konserwacji systemu, a także wprowadzono poprawki w Kontroli konta użytkownikasystemu, aby był mniej inwazyjny. System Windows 7 był również dostarczany ze zaktualizowanymi wersjami kilku standardowych aplikacji, w tym Internet Explorer 8, Windows Media Player i Windows Media Center. W przeciwieństwie do Visty, Windows 7 zyskał uznanie krytyków, a krytycy uznali system operacyjny za znaczną poprawę w stosunku do swojego poprzednika ze względu na lepszą wydajność, bardziej intuicyjny interfejs, mniej wyskakujących okienek Kontroli konta użytkownika i inne ulepszenia wprowadzone na platformie. Windows 7 był wielkim sukcesem firmy Microsoft; jeszcze przed jego oficjalnym wydaniem sprzedaż w przedsprzedaży systemu operacyjnego w sklepie internetowym Amazon.com przekroczyła poprzednie rekordy. W ciągu zaledwie sześciu miesięcy na całym świecie sprzedano ponad 100 milionów kopii, a do lipca 2012 roku liczba licencji wzrosła do ponad 630 milionów. Do stycznia 2018 roku system Windows 10 wyprzedził system Windows 7 jako najpopularniejsza wersja systemu Windows na świecie. Stan na wrzesień 2022 r, 11% tradycyjnych komputerów PC z systemem Windows ma system Windows 7. Windows 11 zajął ostatnio drugie miejsce po Windows 7 jako najpopularniejsza wersja systemu Windows. Wciąż pozostaje popularna w krajach takich jak Syria, Chiny, Indie i Wenezuela.
