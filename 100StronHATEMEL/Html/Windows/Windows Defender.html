Biblioteka
6 stycznia 2005 firma Microsoft udostępniła wersję testową (beta) programu Windows AntiSpyware, który stworzono na podstawie wcześniej zakupionego programu o nazwie Giant AntiSpyware. 14 lutego 2006 nazwę programu zmieniono na Windows Defender, co stało się równocześnie z pojawieniem się drugiej wersji beta programu. Produkt wydano na licencji freeware, zaś jego funkcje to ochrona przed niepożądanym oprogramowaniem, w tym przed programami szpiegowskimi. Użytkownicy systemów Windows XP i Windows Server 2003, których legalność została potwierdzona przez WGA, mogą bezpłatnie pobrać i zainstalować Windows Defendera. Jest on dostępny również jako część systemu operacyjnego Windows Vista i nowszych.
