Windows NT 3.51 – trzecie wydanie systemu operacyjnego Windows NT.

Premiera systemu miała miejsce 30 maja 1995, dziewięć miesięcy po premierze Windows NT 3.5. System posiadał taki sam interfejs użytkownika jak jego poprzednik. Windows NT w wersji 3.51 został określony jako wydanie dla Power PC. Pierwotnie planowano wydać wersję 3.5 dla komputerów Power PC. Edycja 3.51 została wydana dla architektur x86, MIPS i Alpha. Pomimo że NT posiada zupełnie inne jądro, w Windows NT 3.51 możliwe jest uruchamianie aplikacji Win32.

By: Arkadiusz Marzewski, Oliwier Kubka
