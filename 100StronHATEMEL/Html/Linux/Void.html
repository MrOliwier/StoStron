Void to niezależnie opracowany system operacyjny ogólnego przeznaczenia oparty na monolitycznym jądrze Linuksa. Zawiera hybrydowy system zarządzania pakietami binarnymi/źródłowymi, który pozwala użytkownikom szybko instalować, aktualizować i usuwać oprogramowanie lub budować oprogramowanie bezpośrednio ze źródeł za pomocą kolekcji pakietów źródłowych XBPS. Inne funkcje dystrybucji obejmują obsługę komputerów jednopłytowych Raspberry Pi (zarówno armv6, jak i armv7), model rozwoju z codziennymi aktualizacjami, integrację oprogramowania OpenBSD LibreSSL i natywny system init o nazwie „runit”. Void to system operacyjny ogólnego przeznaczenia, oparty na monolitycznym jądrze Linuksa. Jego system pakietów pozwala szybko instalować, aktualizować i usuwać oprogramowanie; oprogramowanie jest dostarczane w pakietach binarnych lub może być budowane bezpośrednio ze źródeł za pomocą kolekcji pakietów źródłowych XBPS. Jest dostępny na różne platformy. Pakiety oprogramowania można budować natywnie lub kompilować krzyżowo za pomocą kolekcji pakietów źródłowych XBPS. Śledź nas na Twitterze, odwiedź kanał IRC #voidlinux na libera.chat i dołącz do subreddita Void Linux. Odwiedź konsolę serwera kompilacji Void, aby uzyskać aktualizacje statusu kompilacji pakietu. Weź udział w projekcie Void Linux, dodając i aktualizując pakiety oraz rozszerzając dokumentację. Więcej informacji można znaleźć w Podręczniku.
